Javascript application designed to provide my userbase the proper type of tea for their current or desired mood. Users are able to input their current mood and 'proper.tea' will return a selection of various teas including brands, reviews, pricing, sources for purchase, and their natural properties. Users are able to create an account, save, update their favorite tea selections, and post reviews to help other users to find their preferred 'cup of tea'.
Python application that displays U.S Maritime shipping volume based on Data.Gov records. Django framework with NVD3 as the visual library. Data was cleaned and then loaded into SQLite3 with Python connector to construct a restfulAPI. NVD3 a javascript library for the Django framework displays graphs of port shipping data.
